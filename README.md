# YSK Ops System v2.0

**PHP + MySQL 企業內部運作系統**  
專為 YSK Limited 設計的管理平台，涵蓋客戶、項目、任務、工時、發票、利潤分析等日常運作。

## 系統特色
- 完全純 PHP + MySQL（無重框架，兼容任何 Web Hosting）
- 專業 UI + Bootstrap 5 + 響應式移動端優化
- 多角色權限 (Admin / PM / Developer / Finance)
- 完整 CRUD + 儀表板 + 統計
- 支援 YSK 四大服務類型 (AI 自動化 / App 開發 / 雲端安全 / Web3 區塊鏈)
- CEO 決策工具 (MRR / 利潤分析 / 客戶貢獻度 / 資源利用

## 主要功能

### 1. 基礎管理
- 用戶管理 (多角色)
- 客戶管理 (CRM + 搜尋 + 狀態)
- 項目管理 (4大服務類型 + 進度 + 預算)
- 任務追蹤

### 2. 運作與財務
- 工時記錄
- 利潤分析 + 預測
- 發票管理 + Stripe 支付
- 週期性發票

### 3. CEO 決策工具
- 完整儀表板 (MRR + KPI)
- 客戶貢獻度報表
- 團隊資源利用率
- AI 項目風險分析

### 4. 規模化功能
- 客戶自助門戶
- 知識庫 + SOP
- 通知中心 + WhatsApp

### 5. UI / UX
- 移動端優化
- Pagination
- 全文搜尋

## 技術棧
- PHP 8+ + PDO
- MySQL
- Bootstrap 5
- Stripe API

## 安裝步驟
1. Clone 或下載 repo
2. 導入 `database.sql`
3. 修改 `config.php` (資料庫 + Stripe Key)
4. 上傳到 Web Hosting
5. 訪問 `index.php` 登入使用
